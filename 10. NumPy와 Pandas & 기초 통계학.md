## 1. 데이터 분석을 위한 자료구조의 변화

일반 프로그래밍과 데이터 분석은 데이터를 대하는 관점이 다릅니다.

* **Python 기본 자료구조 (List, Dict 등):** 다양한 타입의 데이터를 담기 좋지만, 수백만 개의 데이터를 연산하기엔 속도가 매우 느립니다.
* **분석용 자료구조 (Array, DataFrame):** 하나의 데이터 타입만 담는 대신, **벡터화 연산(Vectorized Operation)**을 통해 대량의 데이터를 한 번에 계산합니다. 엑셀의 수식처럼 모든 행에 동시에 연산을 적용하는 원리입니다.

---

## 2. NumPy: 숫자 계산의 심장 (Engine)

넘파이는 고성능 수치 계산을 위한 라이브러리입니다.

### 2.1 데이터의 단일화 (Homogeneous)

넘파이 배열(`ndarray`)은 리스트와 달리 **모든 요소의 데이터 타입이 같아야 합니다.** 만약 섞여 있다면 아래의 우선순위에 따라 강제로 변환됩니다(Upcasting).

> **데이터 타입 강도:** `문자열(str)` > `실수(float)` > `정수(int)` > `불리언(bool)`
> * 예: `[1, 3.14, "A"]` -> 모두 문자열 `"1"`, `"3.14"`, `"A"`로 변환
> 
> 

### 2.2 핵심 함수 및 테크닉

* **`flatten()`**: 다차원 배열을 1차원으로 쭉 폅니다.
* **`reshape()` vs `resize()**`: `reshape`는 모양만 바꿔서 보여주고, `resize`는 원본 배열의 크기를 실제로 바꿉니다. (실무에선 `reshape`를 더 많이 씁니다.)
* **`transpose()`**: 행과 열을 바꿉니다(전치 행렬).
* **불리언 마스킹(Boolean Masking)**: 조건식(예: `data > 10`)을 사용하여 `True`인 데이터만 골라내는 강력한 추출 기술입니다.

## 3. Pandas: 정형 데이터의 관리자 (Manager)

판다스는 넘파이를 기반으로 만들어졌으며, 표(Table) 형태의 데이터를 다루는 데 최적화되어 있습니다.

### 3.1 시리즈(Series)와 데이터프레임(DataFrame)

* **Series:** 엑셀의 '열(Column)' 하나. (인덱스 + 값)
* **DataFrame:** 여러 개의 시리즈가 모인 '표'. 인덱스와 컬럼명을 가집니다.

### 3.2 데이터 접근하기: loc vs iloc (중요!)

| 구분 | 설명 | 사용 예시 |
| --- | --- | --- |
| **`loc`** | **이름(Label)**을 기반으로 조회 | `df.loc[0, 'Name']` |
| **`iloc`** | **인덱스 번호(Integer Position)**를 기반으로 조회 | `df.iloc[0, 1]` |

### 3.3 데이터 탐색의 시작: `describe()`

데이터프레임의 수치형 컬럼들에 대해 **평균, 표준편차, 최솟값, 4분위수** 등 기초 통계량을 한눈에 보여주는 필수 함수입니다.

## 4. 선형(Linear)과 정형(Structured)의 명확한 차이

필기하신 비유를 기술적으로 더 명확히 정리해 드립니다.

* **정형 데이터 (Structured Data):** 데이터가 행과 열의 규격화된 틀(Schema) 안에 들어있는 **상태**를 의미합니다. (엑셀, DB 등)
* **선형 대수 (Linear Algebra):** 데이터를 숫자 덩어리(벡터, 행렬)로 보고, 이들 사이의 수식적 규칙을 찾아 계산하는 **방법론**입니다.
* 즉, **"정형 데이터라는 그릇에 담긴 숫자들을 선형대수라는 도구로 요리한다"**고 이해하시면 좋습니다.

## 5. 기술 통계학과 데이터의 종류

통계는 크게 두 단계로 나뉩니다.

### 5.1 통계의 분류

1. **기술 통계학 (Descriptive Statistics):** 수집한 데이터를 요약, 정리하여 현재 상태를 설명 (평균, 분산 등).
2. **추론 통계학 (Inferential Statistics):** 수집한 표본(Sample)을 바탕으로 전체 모집단(Population)의 특성을 예측.

### 5.2 데이터(자료)의 종류

* **질적 자료 (Qualitative / Categorical):** 숫자로 측정되지 않는 자료
* **명목형:** 순서가 없음 (성별, 혈액형)
* **순서형:** 순서가 있음 (성적 A/B/C, 만족도 상/중/하)


* **양적 자료 (Quantitative / Numerical):** 숫자로 측정되는 자료
* **이산형:** 개수를 셀 수 있음 (자녀 수, 차량 대수)
* **연속형:** 끊김 없이 이어짐 (키, 몸무게, 시간)

## 6. 요약 수치: 데이터가 얼마나 퍼져 있는가?

* **표본 평균 ():** 데이터의 중심 위치.
* **표본 분산 ():** 데이터가 평균으로부터 얼마나 떨어져 있는지의 평균 거리 제곱.
* **표본 표준편차 ():** 분산에 루트를 씌워 실제 데이터와 단위를 맞춘 지표. (값이 클수록 데이터가 넓게 퍼져 있음)
