
# 📘 27일차 TIL: 시계열 데이터와 주성분 분석

### 1. 시계열 데이터의 본질

* **정의:** 일반적인 회귀 분석에 **'시간(t)의 흐름'** 이라는 관점이 더해진 데이터입니다.
* **핵심:** 과거의 데이터가 현재와 미래에 영향을 미친다는 가정을 바탕으로 합니다. ($y_t$는 $y_{t-1}, y_{t-2}, \dots$ 와 상관관계가 있음)

### 2. 시계열 정상성 (Stationarity)

데이터를 분석하기 위한 '전제 조건'으로, 아래 세 가지가 시간에 따라 일정해야 합니다.

1. **평균의 일정:** 평균이 일정하지 않으면 **'차분(Differencing)'** (현재 값에서 이전 값을 뺌)을 통해 정상화합니다.
2. **분산의 일정:** 분산이 일정하지 않으면 **'로그 변환(Log Transform)'** 등을 통해 변동폭을 안정화합니다.
3. **공분산의 일정:** 두 시점 사이의 공분산이 시점 자체가 아닌 '시간 차이'에만 의존해야 합니다.

### 3. 시계열 모델의 종류

* **AR (자기회귀 모형):** 현시점의 자료가 과거의 자료에 의해 설명되는 모델.
* **MA (이동평균 모형):** 현시점의 자료를 과거의 백색잡음(오차)의 선형 결합으로 설명하는 모델.
* **ARIMA (자기회귀 누적 이동평균 모형):** AR과 MA를 합친 모델로, **'누적(Integrated)'** 과정이 포함되어 있어 **비정상 시계열 데이터** 도 차분을 통해 분석이 가능합니다.

### 4. 시계열 자료 분석 방법

* **분석 도구:** AR, MA, ARIMA 등이 있으며, 모델 결정 시 **ACF(자기상관함수)** 와 **PACF(부분자기상관함수)** 그래프를 보고 차수(p, d, q)를 결정합니다.

### 5. 주성분 분석 (PCA)

* **정의:** 여러 변수의 변량을 주성분이라는 서로 독립적인 새로운 변수로 요약하는 차원 축소 기법입니다.
* **학습 포인트:** (기록하신 PAC는 **PCA(Principal Component Analysis)**의 오타일 수 있으니 확인해 보세요!) 데이터의 손실을 최소화하면서 복잡한 데이터를 단순화할 때 사용합니다.
